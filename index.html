Bullet Graphs

The Setup

Stephen Few created the bullet graph in his work creating information dashboards. They are a concise and compact way to visualize how data points match to a goal. In Few's words, bullet graphs visualize "not-to-execeed targets".

Features

  • Specify the minimum and maximum values, low and high range values and the marker value
  • Draw with or without end caps
  • Customize colors to suit

Technology

Silver Bullet is a jQuery plugin. It uses the HTML5 canvas for drawing< and excanvas for IE compatibility.

Compatibility

Silver Bullet has been tested with Firefox 3, Safari 3.1 and IE 7.

The Science

A basic graph

$('.graph').bulletGraph(25, 75, 50);

A huge graph


          $('.hugeGraph').bulletGraph(100, 300, 200, {width: 400, height: 100});

A graph with no edges


          $('.neater').bulletGraph(150, 375, 350, {width: 400, drawEdges: false});

Specifying a graph with POSH


          $('.posh').each(function() {
            var el = $(this);
            el.bulletGraph(el.children('.graph-low').text(),
                           el.children('.graph-high').text(),
                           el.children('.graph-marker').text(),
                           {width: el.children('.graph-max').text()});
            el.children().empty('span');
          });
